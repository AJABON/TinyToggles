# TinyToggles

Illustratorの、人によっては頻繁に切り替える項目「コピー元のレイヤーにペースト」および「プレビュ境界を使用」を簡易的に確認・操作するCEPパネルです。

現在の設定を確認するために毎回レイヤーパネルメニューや環境設定を開くのが面倒だったので作成しました。変更もこちらから可能です。

## インストール

ZXP Installer などを利用してください。
Illustrator再起動後、メニューの ウィンドウ > エクステンション > ペースト先 で出せます。

## できること

* 「コピー元のレイヤーにペースト」の ON/OFF 切り替え
* 「プレビュー境界を使用」の ON/OFF 切り替え
* エクステンション起動時に現在の設定を読み込み、パネルへ反映

## できないこと

* 本来の操作であるレイヤーパネルメニューおよび環境設定から設定を変更した場合、即時の対応更新はできません。ださいので。

## 苦肉の策

ユーザーが標準UIから設定を変更した場合に備え、パネルにマウスカーソルが乗った瞬間に設定を再読込し、表示を更新します。

## 付加機能

Illustrator 30.x以降では、上記に加えて以下の操作時にも自動的にパネルを更新します。

* ドキュメントの切り替え
* コピー

Illustrator 29.x以前では、Adobe CEP Resourcesで配布されている `AIHostAdapter.aip` を別途インストールすることで同様の機能を利用できます。

`AIHostAdapter.aip` は Adobe の配布物であり、本リポジトリには含まれていません。
